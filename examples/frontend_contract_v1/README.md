# Frontend contract v1 example bundle

This directory contains the canonical stable example bundle for external frontend integration.

Included examples:

- `closed_box_minimal.yaml`
- `conical_line_minimal.yaml`

These examples correspond to the stable workflows named in `get_frontend_contract_v1()`.
Any future patch that changes these workflows or replaces these examples must update the frontend contract docs and tests, or explicitly declare `No frontend contract change`.
