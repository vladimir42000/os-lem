# Debug archive

This directory contains historical debug and proof documents.

Use it to preserve:
- diagnostic lineage
- bounded candidate fixes
- investigation checkpoints
- proof branches that informed later decisions

Do **not** use this directory as the primary startup control surface.

Startups should begin from:
- `docs/doc_index.md`
- `docs/start_here.md`
- `docs/current_scope.md`
- `docs/status.md`
- `docs/milestone_charter.md`

Use the debug archive when:
- a current issue clearly reopens an old discrepancy class
- you need to understand why a narrow fix was accepted or rejected
- you need historical compare scripts or output folders

Use `docs/experiment_registry.md` together with this directory.
