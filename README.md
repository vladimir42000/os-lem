# OS-LEM: Open-Source Lumped Element Modeling

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

A Python-based lumped-element modeling kernel for loudspeaker system simulation, inspired by Akabak and Hornresp.

## Features
- Circuit-based electroacoustic simulation
- Modular architecture (parser → solver → postprocess)
- Extensible element library
- Command-line interface and Python API
